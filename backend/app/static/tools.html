Tools — run the pipeline

1. Gmail

Connect the business inbox, then pull recent supplier emails into the document store.

2. Import invoices from Google Drive

Paste the Drive folder link, then Browse to drill into the exact folder you want (e.g. a year or month) and import just that — the most reliable way to get only the invoices you need. Or use the Year box + Import for a quick filter (only works if filenames contain the year). Deduped on Drive file id; safe to re-run. Reconnect Google if you haven't since Drive read access was added.

3. Import bank transactions (BOI CSV)

Upload the Bank of Ireland CSV export(s). They're cumulative and overlap — duplicates are skipped automatically, so it's safe to upload all of them or re-upload. Careys only for now.

4. Run the invoice extractor

Read amounts, references, VAT and statement rows out of imported documents. Run in background processes everything server-side to completion — you can close the tab and it keeps going. The foreground "Run extractor" does up to Max and needs the tab open.

5. Run the matcher

Match a month's bank transactions to their documents and save the suggestions. Safe to re-run.

⚠️ Danger zone — wipe all documents

Deletes every document (email, Drive and archive) for a clean re-import. Cascades to extractions, invoices and document↔transaction links. Bank transactions are kept. Cannot be undone.
